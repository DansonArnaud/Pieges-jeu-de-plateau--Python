# Pieges-jeu-de-plateau--Python
## Description
Ce projet est un jeu de plateau interactif développé en Python avec la bibliothèque FLTK.
Le jeu permet à plusieurs joueurs de s'affronter en déplaçant des tirettes pour faire tomber les billes adverses dans des trous.

## Fonctionnalités
* Interface graphique
* 1 à 4 joueurs
* Génération aléatoire du plateau
* Système de victoire
  
## 🛠️ Technologies
* Python
* FLTK
